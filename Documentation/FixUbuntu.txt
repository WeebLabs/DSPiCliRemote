Follow these instructions to install .NET 10 on Ubuntu and Raspbian
https://learn.microsoft.com/en-us/dotnet/iot/deployment

Then, 
On Ubuntu you have to allow access to the USB port for it to work. 
The simplest approach is to run these lines:

# Create a group called usbwriters
sudo groupadd usbwriters
# Add you to the group
sudo usermod -a -G usbwriters $USER
# Allow access to the USB port by adding a rule granting our group access to the subsystem
echo 'SUBSYSTEM=="usb", MODE="0666", GROUP="usbwriters"' | sudo tee /etc/udev/rules.d/99-usbwriters.rules
# Reboot
echo "Please reboot"
